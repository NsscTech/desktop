### 启动服务

```
docker-compose -f docker-compose.yml up -d
```

### 查看服务

```
docker-compose -f docker-compose.yml ps
```

### 关闭服务

```
docker-compose -f docker-compose.yml down
```

### 重启服务

```
docker-compose -f docker-compose.yml restart
```
